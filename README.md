# FiveM Racing Server

A QBCore-based FiveM server featuring racing mini games with vehicle selection.

## Quick Start

1. Set up a VPS with Ubuntu 22.04
2. Install FiveM server artifacts
3. Install QBCore framework
4. Add racing resources
5. Configure `server.cfg`
